# ⬡ DCode — AI Code Editor

> Simple Python code editor with AI. Works on Windows, Mac, Linux.
> One file. No complex setup.

---

## ⚡ Install & Run (3 steps)

### Step 1 — Install Python
Download from **python.org** → Install  
(Python 3.8 or newer)

### Step 2 — Install AI library (optional)
```bash
pip install anthropic
```

### Step 3 — Run DCode
```bash
python dcode.py
```

**That's it!** App opens automatically.

---

## 🎯 Features

| Feature | Description |
|---|---|
| **12+ Languages** | Python, JS, TS, HTML, CSS, Java, Go, Rust, C++, PHP, SQL, Ruby |
| **Syntax Highlight** | Color coding for all languages |
| **Real Code Run** | F5 runs Python code instantly |
| **Claude AI** | Explain, fix, optimize, generate code |
| **🎙 Voice Input** | Say "define function called greet" → gets typed! |
| **Snippets Library** | 30+ ready-made code snippets |
| **1-Click Libraries** | NumPy, Pandas, sklearn, PyTorch examples |
| **Auto-indent** | Smart indentation on Enter |
| **Auto-brackets** | () [] {} "" '' auto-close |
| **Find & Replace** | Ctrl+F |
| **pip Install** | Install packages from inside editor |
| **Git Status** | See git status in terminal |
| **Dark/Light** | Two themes |

## 🔑 Add Claude AI (optional but powerful)

1. Go to **console.anthropic.com**
2. Sign up (free)
3. Create API key
4. In DCode: **AI menu → Set API Key**
5. Paste key → Activate

## ⌨️ Shortcuts

| Key | Action |
|---|---|
| F5 | Run code |
| Ctrl+S | Save |
| Ctrl+N | New file |
| Ctrl+O | Open file |
| Ctrl+E | AI Explain |
| Ctrl+B | AI Fix Bug |
| Ctrl+G | AI Generate |
| Ctrl+M | 🎙 Voice Input |
| Ctrl+/ | Toggle comment |
| Ctrl+D | Duplicate line |
| Ctrl+= | Font bigger |
| Ctrl+- | Font smaller |
